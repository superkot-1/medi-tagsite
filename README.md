# MediTag - Медицинский Браслет ID

## Описание
MediTag - это инновационная система медицинской идентификации, которая обеспечивает мгновенный доступ к критически важной медицинской информации в чрезвычайных ситуациях через QR-коды и уникальные номера тегов.

## Функции
- Регистрация медицинских браслетов с уникальными номерами
- Управление личной медицинской информацией
- Мгновенный доступ для спасателей через QR-коды
- Безопасное хранение данных с шифрованием уровня банков
- Соответствие стандартам HIPAA

## Структура Сайта

### Главные Страницы
- `index.html` - Главная страница с регистрацией браслета
- `about.html` - Информация о компании и услугах
- `contact.html` - Контакты и поддержка
- `cabinet.html` - Личный кабинет пользователя

### Ресурсы
- `resources/` - Изображения и медиа-файлы
- `main.js` - Основной JavaScript файл
- `design.md` - Документация дизайна
- `interaction.md` - Документация взаимодействия

## Используемые Технологии
- HTML5 + CSS3 (Tailwind CSS)

## Установка и Запуск

### Локальный Запуск
1. Клонируйте или скачайте файлы
2. Перейдите в директорию проекта
3. Запустите локальный сервер:
```bash
python3 -m http.server 8000
```
4. Откройте браузер и перейдите на `http://localhost:8000`

### Проверка Синтаксиса
```bash
# Проверка HTML
node -c main.js

# Проверка доступности ресурсов
ls -la resources/
```
